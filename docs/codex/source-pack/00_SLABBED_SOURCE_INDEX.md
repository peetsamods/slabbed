## Supersession rule

Any old source that says slab support is selective-only, that category expansion is currently safe by default, or that a savepoint-pending fix is final/fixed is superseded.

Current Slabbed intent is global slab support through canonical slab-state grammar, not global solidity lies and not disconnected rescue patches.
